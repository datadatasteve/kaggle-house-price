# House Prices - Quick Data Exploration

## Project Goal
A quick warmup project to practice data analysis fundamentals before tackling Pittsburgh real estate analysis.

## Objectives
- Load and explore the Ames, Iowa housing dataset
- Clean and understand the data structure
- Create visualizations showing price relationships
- Document findings and insights
- Practice Git workflow

## Dataset
Kaggle House Prices: Advanced Regression Techniques
- 1,460 homes in Ames, Iowa
- 79 features including square footage, quality ratings, location, etc.
- Target: Sale price

## Skills Practiced
- Python/pandas data manipulation
- Data cleaning
- Matplotlib/Seaborn visualization
- Markdown documentation
- Version control with Git
